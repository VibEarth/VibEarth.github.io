# ESP32 프로젝트 - Intro

## ESP32란?

ESP32는 Espressif사의 소형 마이크로컨트롤러 칩을 활용한 개발보드를 통칭하는 말입니다.
교육용으로 널리 쓰이는 아두이노 계열이라고도 할 수 있는데,
아두이노 개발환경과 라이브러리를 그대로 사용할 수 있기 때문입니다.

아두이노처럼 ESP32에도 다양한 칩과 개발보드가 있습니다.
2026년 현시점에서 가성비가 좋은 **ESP32-C3**와 고성능인 **ESP32-S3**를 비교하면 아래와 같습니다.


| 항목 | ESP32-C3 | ESP32-S3 |
|------|----------|----------|
| CPU | RISC-V 싱글코어 160MHz | Xtensa LX7 듀얼코어 240MHz |
| RAM | 400KB SRAM | 512KB SRAM (+PSRAM 최대 8MB) |
| WiFi | Wi-Fi 4 (2.4GHz) | Wi-Fi 4 (2.4GHz) |
| Bluetooth | BLE 5.0 | BLE 5.0 |
| GPIO | 16~22개 | 45개 |
| USB | Serial | OTG |
| AI 가속 | 없음 | 벡터 확장 지원 |
| 특징 | 저전력, 소형, 저가 | 고성능, 카메라·AI 지원 |
| 가격대 | 3,400원~ | 9,000원~ |

---

## 왜 ESP32인가?

그동안 학교 현장에서는 피지컬 컴퓨팅 교육을 위해 아두이노 우노 호환보드를 많이 사용해왔습니다.
만 원 이내의 저렴한 가격에 손바닥보다 작은 크기로 다양한 기능과 라이브러리를 활용할 수 있기 때문입니다.

ESP32C3는 우노보다 더 작으면서도 **WiFi와 블루투스 기능을 기본 제공**합니다.
아두이노 우노는 센서값을 확인하려면 LCD 화면을 연결하거나 컴퓨터에 연결해야 하지만,
ESP32는 화면도, 컴퓨터도 필요 없이 **WiFi로 스마트폰에 바로 연결**하여 결과값을 확인할 수 있습니다.

ESP32, 작지만 강합니다. **ESP32, Tiny but Powerful ⊕**

---

## 무엇을 만드나요?

ESP32에는 다양한 기상 센서를 연결할 수 있습니다.
온도, 습도, 기압, 미세먼지, 가스 센서 등을 연결하여
대기질을 측정하고 모니터링하는 **소형 기상관측기**를 만들어보려고 합니다.

이를 위해 ESP32의 기본 LED 예제부터 시작하여
각 기상 센서의 예제 코드를 하나씩 실습합니다.
그리고 실습한 센서들을 종합하여 **ESP32 소형 기상관측기**를 완성해보겠습니다.
