    ESP32 Project - C3 SuperMini - Vibe for Settings

    C3 SuperMini 설정을 위한 바이브🜨

    요즘 IT, 코딩 관련 책이나 블로그 등을 볼 때, 가장 빠르게 넘기는 부분이 설정 부분이 아닐까 싶습니다. 최근에는 생성형 AI의 도움을 받으면 기본 설정을 손쉽게 따라할 수 있고, 시간을 조금만 들이면 트러블 슈팅도 충분히 할 수 있기 때문입니다. Vib⊕arth답게 주의해야할 사항 몇 가지만 소개합니다.

    1. 아두이노 IDE 설치 + 보드 매니저에서 ESP32C3 선택

    아두이노 IDE 설치는 공식 다운로드 페이지를 따라가시면 됩니다. 윈도우와 맥os 버전 모두 있습니다.

    아두이노 IDE의 보드매니저에서 ESP32를 검색하면 비슷한 이름이 여러 개 뜹니다.

    • ESP32C3 Dev Module
    • ESP32-C3-DevKitM-1
    • 그 외 변형들

    가장 범용적으로 쓰이는 ESP32C3 Dev Module을 선택하시면 됩니다.

    2. USB-C 연결 + 포트(Port) 확인

    보드를 컴퓨터에 연결 한 후 IDE 좌측 상단(또는 도구 > 포트)에서 보드가 잡히는지 확인합니다.

    초보자 체크포인트: "포트"가 뭔지 모르는 단계라면 여기서부터 막힐 수 있습니다.

    • 도구(Tools) 메뉴 → 포트(Port)에서 COM3, /dev/cu.usbserial- 같은 이름이 보이면 정상 인식된 것입니다.
    • 여기서 주의할 점은, 컴퓨터가 기본적으로 잡고 있는 포트가 있다는 것입니다. 따라서 C3보드를 연결할 때 새로운 포트가 생기는지가 중요합니다.

    드라이버 확인: SuperMini는 구매처/제조사에 따라 USB-시리얼 칩셋이 다릅니다. 흔히 CP2102 계열과 CH340/CH341 계열이 쓰이는데, 칩 종류에 따라 필요한 드라이버가 다릅니다. 포트가 아예 안 보인다면 보드의 칩 각인을 확인해서 어떤 계열인지 먼저 파악하는 게 빠릅니다.

    • CP2102 드라이버 다운로드
    • CH340/CH341 통합 드라이버 (CH341SER) — 이름은 CH341이지만 CH340 계열도 함께 지원

    IDE에서 코드를 ESP32C3 보드로 업로드(전송 및 저장)하기 위해서는 반드시 보드가 연결된 포트로 선택되어 있어야 합니다.

    3. 업로드 모드 — BOOT / RESET 버튼

    아두이노, ESP32를 처음 접할 때 가장 많이 헤맬 수 있는 지점입니다.

    SuperMini C3에는 작은 버튼 두 개가 있습니다 — BOOT와 RESET

    • 자동 업로드 회로가 있는 보드는 버튼 없이도 업로드가 됩니다.
    • 하지만 가끔 자동 인식이 실패하거나, 한 번 꼬이면 수동으로 부트 모드에 진입해야 합니다.

    수동 부트 모드 진입 순서:

    1. USB-C 케이블이 연결된 상태에서 BOOT 버튼을 누른 채로 유지합니다.
    2. BOOT 버튼을 누른 채로 RESET 버튼을 짧게 눌렀다 뗍니다. (리부팅되는 순간 BOOT 상태를 읽어 업로드 모드로 진입합니다)
    3. BOOT 버튼을 놓습니다.
    4. 이 상태에서 업로드를 시작하면 이미 업로드 모드이므로 바로 진행됩니다.

    핵심은 USB를 새로 연결하는 게 아니라, 리셋이 일어나는 순간 BOOT 버튼이 눌려있는가입니다. ESP32는 리셋(리부트)하는 타이밍에 BOOT핀 상태를 읽어서 모드를 결정하기 때문에, RESET 버튼으로 리부팅하는 쪽이 USB를 뽑았다 꽂는 것보다 훨씬 안정적입니다.

    4. 보드레이트(Baud Rate) 확인

    시리얼 모니터를 열었는데 글자가 깨지거나 아무것도 안 뜬다면 보드레이트 불일치를 의심해봅니다.

    • 일반적으로 코드에서 Serial.begin(115200);을 많이 쓰지만
    • 가끔 라이브러리 기본값이나 예제 코드가 9600으로 되어 있는 경우가 있습니다.
    • 시리얼 모니터 우측 하단의 보드레이트 드롭다운을 코드와 맞춰줍니다.

    5. C3 SuperMini 데이터시트 / 확장보드 핀아웃 훑어보기

    앞으로 계속 보게 될 핀맵이니 지금 미리 한 번 눈으로 익혀두면 나중에 편합니다.

    • ESP32-C3 SuperMini 데이터시트, 핀아웃
    • C3 SuperMini 확장보드

    우리는 PCB개발자가 아니라 보드 사용자이므로 데이터시트, 핀아웃, 스케마틱을 모두 이해하고 있을 필요는 없습니다. "보드에 대한 부품과 회로가 잘 정리된 문서가 있구나" 정도만 기억하시고 나중에 AI에게 링크나 pdf파일 주고 필요한 정보 알려달라고 하면 잘 정리해줍니다.
